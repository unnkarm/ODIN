System Architecture

Six layers.
One mind.

ODIN's intelligence stack is built in six tightly integrated layers — from raw space-weather ingestion through physics simulation, AI decision-making, and full explainability logging.

Intelligence Stack

Built layer
by layer.

Each layer is purpose-built for a specific class of computation — from fetching real-world data to generating and ranking AI alternatives in milliseconds.

01
Data Layer
Real-world ingestion from NASA, NOAA, and orbital debris feeds. Timestamps map to historical space weather archives — solar flux, particle storm intensity, and debris density fields.
Reality Engine
02
Trajectory Brain
2-body orbital mechanics computing Earth–Moon Hohmann transfers. Optimizes departure windows, burn durations, and coast arcs. Outputs trajectory JSON with per-timestep state vectors.
Physics Engine
03
Risk Brain
Continuous monitoring loop reading solar flare intensity, debris collision probability, and navigation error accumulation. Classifies threats against configurable GREEN/YELLOW/RED thresholds.
Threat Classifier
04
Path Planner
A* and Dijkstra search over discretized trajectory space. Cost function: C = 0.30·fuel + 0.25·time + 0.35·risk + 0.10·success. RRT heuristics handle dynamic constraints during replanning.
A* / RRT
05
AI Copilot
LLM-powered decision layer generating multi-option replanning strategies with natural-language rationale. Prompt-engineered to reason over hazard type, constraints, and mission priority weights.
Generative AI
06
Explainability Logger
Full reasoning chain logged for every autonomous decision. Human-readable audit trail: what was detected, how options were scored, what was chosen, and why. XAI transparency by design.
XAI / Audit
